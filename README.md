# InsureGuard-Advanced-Claim-Risk-Modeling-Feature-Discovery
The dataset utilized in this project comprises historical data on insurance claims, encompassing diverse information about policyholders, their demographics, past claim history, and related policy attributes. It is structured to facilitate predictive modeling tasks aimed at identifying the probability of future claims.

Objective: Develop a predictive model that identifies the likelihood of a policyholder making a claim, while handling class imbalance effectively.
